# Romantic Anniversary

Pagina romantica de aniversario hecha con HTML, CSS y JavaScript.

Incluye:

- Portada con foto y contador desde el inicio de la relacion.
- Musica de fondo.
- Carta animada con efecto de escritura.
- Invitacion protegida con contrasena basada en la fecha especial.
- Video de invitacion.
- Mensaje final con confirmacion directa por WhatsApp.

## Estructura

```text
romantic-anniversary/
├─ index.html
├─ css/
│  └─ style.css
├─ js/
│  └─ script.js
└─ assets/
   ├─ audio/
   ├─ images/
   └─ video/
```

## Como verlo

Abre `index.html` directamente en el navegador.

## Publicar en GitHub Pages

1. Sube el proyecto a un repositorio de GitHub.
2. Entra a `Settings`.
3. Ve a `Pages`.
4. En `Branch`, selecciona `main` y la carpeta `/root`.
5. Guarda los cambios.

GitHub generara un enlace publico para ver la pagina.

## Nota

El video esta dentro de `assets/video/`. GitHub permite archivos de hasta 100 MB, asi que conviene mantenerlo optimizado.
